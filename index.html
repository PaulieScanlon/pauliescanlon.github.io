A dashboard for GitHub Events

This site is built using Astro and displays both static and dynamic GitHub user and repository events data.

Data is sourced at both build and run-time using Cross Origin Gatsby Functions provided by Paulie API

The src code for this project can be found here: PaulieScanlon/pauliescanlon.github.io
